Steps for running the Enumerate.java file:

1. Compile and Run the Enumerate.java file using the below commands:
		> javac Enumerate.java
		> java Enumerate
				OR
		> java Enumerate {Optional 2 integers as command line arguments}
    Note: The parameters can be changed statically from the file as well.

2. Alternatively it can be run in Intellij/ Eclipse or any Java Compatible IDE.


Steps for running the EnumerateTopological.java file:

1. Compile and Run the Enumerate.java file using the below commands:
		> javac EnumerateTopological.java
		> java EnumerateTopological
